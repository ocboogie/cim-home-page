Key points for jQuery....

  • Need to first load a specific jQuery versions using a CDN
  • The jQuery object is then available, it uses the identifier name $.
  • When used in an identifier, $ is treated as a letter, but it is reserved for special purposes.
  • Other typical identifiers (aka variable names) are: myHightlight, bg, fg, $, str, border.
  • A jQuery call consists of the $( jQuerySelector ).methodName
  • Where the jQuerySelector is a string based on standard CSS3 selectors
  • Basic CSS selectors can be a tag name such as li, ul, body, etc.
  • CSS selectors can also reference the HTML id attribute of a tag using #idName
  • CSS selectors can also reference a group of tags, using the HTML class atrribute using .className
